Overview

An HTTP backend for Riak::Client that uses the ‘curb’ library/gem. If the ‘curb’ library is present, this backend will be preferred to the backend based on Net::HTTP. Conforms to the Riak::Client::HTTPBackend interface.

def initialize(client)
  super
  @curl = Curl::Easy.new
  @curl.follow_location = false
  @curl.on_header do |header_line|
    @response_headers.parse(header_line)
    header_line.size
  end
